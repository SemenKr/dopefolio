Давайте познакомимся!

Я активно учусь и постоянно расширяю свой арсенал навыков. Среди моих основных компетенций — знание HTML, CSS и JavaScript. Я также знаком с технологиями Pug, SCSS и системой контроля версий Git. Мой опыт работы включает использование WordPress для разработки веб-сайтов.

FindEdu

Это проект, разработанный для продажи онлайн-курсов. Он выполнен с использованием технологии Pug для удобного шаблонизирования HTML. Сайт включает в себя инновационные функции, такие как слайдеры с использованием no ui slider для улучшения пользовательского опыта и навигации по курсам. Особое внимание уделено адаптивной верстке, чтобы пользователи могли комфортно просматривать сайт на различных устройствах — от компьютеров до мобильных телефонов.
